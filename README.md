# 🔬 AI Startup Validator

> Валидация бизнес-идей через AI Customer Discovery по методологии **Mom Test**

---

## 📌 Что это такое

Инструмент проводит симуляцию custdev-интервью с виртуальными клиентами, анализирует результаты и выдаёт рекомендации — за несколько минут, без затрат.

Используется как быстрая **предварительная проверка** идеи перед реальными интервью и разработкой.

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🧠 **AI Customer Discovery** | Генерация персон и интервью по Mom Test через Groq LLM |
| 📊 **Живые бенчмарки** | Актуальные отраслевые данные через DuckDuckGo + LLM-синтез |
| 💡 **Рекомендации** | Автоматические выводы на основе результатов интервью |
| 🏆 **Итоговый Score** | Оценка жизнеспособности идеи от 0 до 100 |
| 🌍 **Локализация** | Русский и казахский язык, адаптация под страну |

---

## 🛠 Технический стек

- **Backend**: Python 3.11+, Streamlit, Pydantic v2
- **LLM**: Groq API — `llama-3.3-70b-versatile`
- **Поиск**: DuckDuckGo Search
- **Визуализация**: Plotly, Pandas

---

## 🚀 Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Make-09/AI-startup-validator.git
cd AI-startup-validator

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Получить API ключ Groq (бесплатно)
#    https://console.groq.com/ → API Keys → Create API Key

# 4. Добавить ключ в переменные окружения
export OPENAI_API_KEY=your_groq_key_here

# 5. Запустить
streamlit run main.py
```


## 📋 Как пользоваться

1. Откройте приложение
2. Заполните описание идеи, целевую аудиторию и цену
3. Выберите модель монетизации (подписка / pay-per-use / комиссия)
4. Нажмите **«🚀 Запустить валидацию»**
5. Изучите результаты интервью и рекомендации

> **Лимит:** 20 запусков в сутки — общий для всех пользователей сервиса. Сбрасывается в полночь по Алматы (UTC+5).
