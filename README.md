# 🔬 AI Customer Discovery Validator

AI-симуляция customer development интервью по методологии Mom Test для предварительной валидации бизнес-идей.

## Возможности

- **3 Модели монетизации** — поддержка ежемесячной подписки (SaaS), разовой оплаты (Pay-per-use) и комиссии со сделок (Marketplace).
- **AI Customer Discovery** — динамическая генерация персон и интервью по методологии Mom Test через Groq (llama-3.3-70b-versatile) с защитой от искажений.
- **Живые бенчмарки (2025–2026)** — система ищет актуальные отраслевые и страновые бенчмарки через DuckDuckGo Search и синтезирует их через LLM.
- **Рекомендации** — автоматическая генерация выводов на основе результатов интервью.
- **Локализация** — поддержка русского и казахского языков, адаптация под страну (Казахстан, Россия, США, Global).

## Технический стек

- **Python 3.11+**, Streamlit, Pydantic v2
- **LLM**: Groq API — llama-3.3-70b-versatile
- **Визуализация**: Plotly, Pandas
- **Поиск бенчмарков**: DuckDuckGo Search

## Установка и запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Получить бесплатный API ключ Groq
# Перейти на https://console.groq.com/ → API Keys → Create API Key

# 3. Запустить приложение
streamlit run main.py

# 4. Ввести API ключ в боковой панели приложения
```

## Структура проекта

```
├── models.py              — Pydantic-модели
├── benchmarks.py          — живые бенчмарки (DuckDuckGo + LLM) и статические (fallback)
├── personas.py            — динамическая генерация персон через LLM
├── interview_engine.py    — AI-интервью (Mom Test)
├── aggregator.py          — количественная агрегация результатов
├── insights.py            — генерация рекомендаций на основе интервью
├── main.py                — Streamlit UI (3 вкладки)
└── requirements.txt
```

## Использование

1. Заполните описание бизнес-идеи, целевую аудиторию и ценовую политику
2. Введите API ключ Groq в боковой панели
3. Нажмите «Запустить валидацию»
4. Просмотрите результаты AI-интервью и рекомендации
5. Получите общую оценку жизнеспособности (0–100)
